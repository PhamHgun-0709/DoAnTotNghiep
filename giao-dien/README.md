# Frontend Interface (Giao Dien)

UI project for campaign monitoring and optimization insights.

## Structure
- `public`: Static assets.
- `src/pages`: Screen-level pages (dashboard, campaign detail, recommendation).
- `src/components`: Reusable UI components.
- `src/layouts`: App layout wrappers.
- `src/services`: API client and data fetching logic.
- `src/styles`: Global and module styles.
- `src/utils`: Frontend utility helpers.
